Symptom

After you navigate into different pages, the page you visited before looks different than before.

Use CSS Nesting

When you define the css style, you can use it under the element which can be used to distinguish it.
I'm a normal black div
I'm a normal black div, too
    .CSSConflict .page-section .inner-section .css-conflict-div-1 {
        border: 3px gray solid;
        margin: 10px;
        width : 300px;
        height: 100px;
        background-color: black;
    }
    .inner-section .css-conflict-div-2 {
        border: 3px gray solid;
        margin: 10px;
        width : 300px;
        height: 100px;
        background-color: black;
    }
