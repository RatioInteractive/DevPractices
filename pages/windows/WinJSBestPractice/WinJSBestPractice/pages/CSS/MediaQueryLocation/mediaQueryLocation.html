Symptom

Defined media query for overriding some style does not work.

Define the media query at the end of CSS file

We usually define media query with some style we want to override. Because it's overriding, it should be defined after the original style is defined.
I'm supposed be blue when the screen got smaller. My media query is defined at the beginning of the css file.
I'm supposed be blue when the screen got smaller. My media query is defined at the end of the css file. 
    .mediaQueryLocation .page-section .media-query-location-top {
        background-color: blue;
        width: 300px;
    }
    .mediaQueryLocation .page-section .media-query-location-bottom {
        background-color: blue;
        width: 300px;
    }
